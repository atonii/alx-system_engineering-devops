## Shell Redirections ##

[0-hello_world](https://github.com/atonii/alx-system_engineering-devops/blob/master/0x02-shell_redirections/0-hello_world) - Prints "Hello, World" to the display

[1-confused_smiley](https://github.com/atonii/alx-system_engineering-devops/blob/master/0x02-shell_redirections/1-confused_smiley) - Print confused smiley using the escape character "\"

[2-hellofile](https://github.com/atonii/alx-system_engineering-devops/blob/master/0x02-shell_redirections/2-hellofile) - Display content of file



[3-twofiles](https://github.com/atonii/alx-system_engineering-devops/blob/master/0x02-shell_redirections/3-twofiles) - Display content of two files

[4-lastlines](https://github.com/atonii/alx-system_engineering-devops/blob/master/0x02-shell_redirections/4-lastlines) - Display last 10 lines of a file

[5-firstlines](https://github.com/atonii/alx-system_engineering-devops/blob/master/0x02-shell_redirections/5-firstlines) - Display first 10 lines of a file

[6-third_line](https://github.com/atonii/alx-system_engineering-devops/blob/master/0x02-shell_redirections/6-third_line) - Print Nth line of file. The usages are tail -n+N | head -1: 3.7 sec,
head -N | tail -1: 4.6 sec, sed Nq;d: 18.8 sec

7-file - uses escape character to name a file with special characters
